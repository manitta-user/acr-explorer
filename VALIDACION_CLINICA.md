# Validación clínica (plantilla)

Este documento registra la **revisión por un profesional médico** de una muestra
de las recomendaciones y traducciones del proyecto. Sirve para dar respaldo y
transparencia (no constituye validación regulatoria de dispositivo médico).

## Cómo completarla

1. Tomar una muestra aleatoria de **30 mapeos Dx → estudio** y **30 traducciones**
   (procedimientos y/o escenarios).
2. Un médico/radiólogo evalúa cada ítem como **Correcto / Aceptable / Incorrecto**.
3. Registrar el **% de acuerdo** y los comentarios.
4. Firmar.
